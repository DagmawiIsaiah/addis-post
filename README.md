News website that provides news from all around the world.

## Installation

Build from source code.
- Clone the repo

```sh
git clone https://github.com/DagmawiIsaiah/addis-post.git
```
- And then you can use the normal build & run procedure 
```sh
flutter pub get
flutter run
```
